# Manual Github #






#### Clonar un repositorio externo (origin). ####
Desde GitHub vamos al repositorio que queremos clonar, vamos a Code/copy repository
A continuacion vamos a crear un nuevo proyecto y copiamos la url del repositorio que queremos clonar

Una vez clonado ya podemos crear el archivo readme.md como prueba y hacer un commit

El siguiente paso es crear un nuevo repositorio desde GitHub, una vez creado vamos entramos en el y copiamos el enlace del repositorio
De vuelta en la ventana de proyecto vamos a Git/Manage Remotes
Eliminados la rama que esta creada y creamos una nueva con la url que copiamos desde el repositori
